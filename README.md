## 📌 Overview

Complete **Data Analysis Assignment** on **20 leading hospitals from around the world** — including Mayo Clinic, Cleveland Clinic, Johns Hopkins, AIIMS, Aga Khan University Hospital, and Shaukat Khanum Memorial Cancer Hospital. Analyzes bed capacity, staff strength, annual patients, specialties, ratings, and budgets across 15 countries.

## 📊 Dataset Features

| Column | Description |
|--------|-------------|
| `Hospital_Name` | Name of the hospital |
| `Founded` | Year founded |
| `City` / `Country` | Location |
| `Beds` | Total bed capacity |
| `ICU_Beds` | Intensive Care Unit beds |
| `Doctors` | Number of doctors |
| `Nurses` | Number of nurses |
| `Annual_Patients` | Patients treated per year |
| `Emergency_Cases_Year` | Emergency cases handled per year |
| `Specialties` | Number of medical specialties offered |
| `Type` | Public / Private |
| `Rating_Out_Of_5` | Overall rating |
| `Annual_Budget_USD` | Annual operating budget in USD |

## 🚀 How to Run

### Google Colab
1. Go to [colab.research.google.com](https://colab.research.google.com)
2. **File → Upload notebook** → select `Hospital_Analysis.ipynb`
3. **Runtime → Run All**

### Local
```bash
pip install pandas matplotlib seaborn plotly
jupyter notebook Hospital_Analysis.ipynb
```
